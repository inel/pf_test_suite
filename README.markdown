# ProgForce test Suite

#Installation

* Install rvm : 
bash -s stable < <(curl -s https://raw.github.com/wayneeseguin/rvm/master/binscripts/rvm-installer)

* Create gemset : 
rvm use 1.9.2@testproj --create

* Instal bundler :
gem install bundler

* update all gems : 
bundle install
